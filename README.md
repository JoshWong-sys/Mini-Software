Project Title: Mini Task Manager System

Technology Used: Python Console Application

Feature Description:
The Task Manager allows users to manage daily tasks. Users can create new tasks, update existing tasks, remove completed tasks, and display all current tasks. The system uses a simple menu-driven interface and stores task information temporarily using a Python list.

Main Functions:

Add new tasks
Edit existing tasks
Delete tasks
Display task list

Future Improvements:

Save tasks to a database
Add user accounts
Add task deadlines and priorities
